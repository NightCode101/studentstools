After deploying to Vercel, you'll get nameserver addresses like:

### Step 3: Update Domain DNS at Registrar
1. Go to your domain registrar (GoDaddy/Namecheap)
2. Find "DNS Settings" or "Nameservers"
3. Replace with Vercel's nameservers
4. Save changes
5. **Wait 24-48 hours** for DNS propagation

### Step 4: Add Domain to Vercel
1. Go to Vercel Dashboard
2. Select your project
3. Go to Settings → Domains
4. Enter: `studentstools.site`
5. Add `www.studentstools.site` (optional)
6. Vercel will verify automatically

### Step 5: Test It Works
- Visit `https://studentstools.site`
- Should load your site instantly
- Check SSL certificate (green lock icon)

## 🚨 Common Issues

### DNS not updating?
- Wait full 48 hours
- Try different DNS checker: https://dnschecker.org/

### Vercel shows SSL error?
- Make sure DNS is fully propagated
- Force refresh: Ctrl+Shift+Delete (cache)
- Try incognito window

### Email not reaching me?
- Check spam folder
- Verify forwarding address in Vercel
- Wait 10 minutes after setup

### Site still showing Vercel subdomain?
- Verify DNS propagated
- Clear all caches
- Restart browser

### Can't add domain to Vercel?
- Make sure you own the domain
- Verify nameserver points to Vercel
- Try removing and re-adding
